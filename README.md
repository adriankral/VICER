DEPLOYMENT

1. Install Node.js >= 0.8.0
2. Update node to the latest version ->npm update -g npm
3. Intall grunt CLI ->npm install -g grunt-cli
4. Go to the project root.
5. Install dependencies ->npm install
6. Run project ->grunt
